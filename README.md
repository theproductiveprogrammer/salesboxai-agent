# SalesboxAI Agent - Local AI Assistant

SalesboxAI Agent is an AI assistant that helps increase the productivity of Sales people and Marketing agents by providing them with an AI agent that can help them with prospecting and reach out.

Your SalesboxAI Agent can:

* Discover Leads
* Engage on multiple channels (Linkedin/Email/...)
* Set up meetings
* Send personalized messages
* Coordinate multi-touch campaigns
